# Expressive code

> Notez les 3 principales idées à retenir à propos du Mob Programming ?

1. Répartition en rôles entre les membres du groupe tels que Driver, Navigator et Mob afin de varier les idées et façon de travailler.
2. Plus on est nombreux et plus ça prend de temps pour avancer.
3. Réduire la complexité 


Plus le navigateur donne des directives de haut niveau, plus ça donne de liberté au conducteur.
- Intention   (Ex: Acheter un paquet de pattes)
- Location    (Ex: A ce magasin)
- Detail      (Ex: De cette marque)
